\subsection{Plug and Play Language Models}
We build off the plug and play language model (PPLM) framework presented in \cite{Dathathri2020Plug}, which frames the controlled generation problem as generating from a language model $p(x)$ by taking a backward step in the hidden state along the gradient for a style discriminator $d_s$ on $x$. Our work expands PPLM in two ways: (i) we take a backward step along the gradient for multiple style discriminators, and (ii) we adjust the gradient from the language model along the gradient from one discriminator, rather than taking a full backward step. These adjustments are  in two ways: (i) a gradient weighting by discriminator score is used to adjust the language model's hidden state along the gradient of only the target style's discriminator, and (ii) the gradient from the  discriminators is adjusted by a scalar.

\subsection{Reinforcement Learning}
We use reinforcement learning to fine-tune language models to produce outputs with particular styles. Given a language model $p(x)$ and a reward function $R(x)$ for the target style, we fine-tune the model using reinforcement learning. The gradient for the loss of the model is adjusted by the gradient for $R(x)$.
