\subsection{Graph Data Challenges with LLMS}
LLM can not directly handle graph data, as graph data are unorganized and complex. Secondly, there is a wide range of graph-related tasks, designing efficient input format for different tasks and effective prompt techniques is essential while rarely explored. LLMs have demonstrated some capability in handling graph-structured data, there remains a substantial need for further devel- opment to achieve a performance level comparable to specialized graph-oriented model
