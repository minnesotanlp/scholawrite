These results are consistent with findings on the importance of graph structure and visual representation in graph reasoning tasks by \cite{li2023deepfake}. Notably, our findings indicate that the choice of encoding modality is a significant factor in determining the effectiveness of image modality encoding in graph task, with factors such as edge encoding function, graph structure, and image representation being key to determining the effectiveness of the image modality.

We also find that in graphs, the node classification task is affected by the graph structure, the nature of the encoding function used, and the image representation. In particular, graphs with a high number of edges influence the performance of node classification using image modality. Similarly, graphs with a greater number of motifs have a greater impact on node classification using image modality. This is because the motif structure of a graph can significantly affect the image representation of the graph, making it easier or more difficult to understand the graph structure. 

We also find that in graphs, the node classification task is affected by the encoding function used to represent the graph. For example, when the graph is represented as a text encoding, the task performance is dependent on the nature of the encoding function used. Figure \ref{fig:fig2} illustrates that using the Adjacency List as the mode of encoding for the graph results in higher accuracy in node classification. This is because the Adjacency List provides the LLM, the local context of the linked nodes. As the number of edges (or ``linked nodes'') increases, the performance of node classification using the image modality decreases. This is because the LLM can only store so much information about the nodes at a given node. Thus, if the node is connected to many other nodes, it can be difficult for the LLM to determine which labels are being shared among the nodes. Therefore, if the task is too complex, the image modality is not the best for encoding the graph.
