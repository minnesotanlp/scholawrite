\subsection{Domain-Based Performance}
When considering each subsection of the evaluation set in Table~\ref{tab:precision_domain}, we observe that the Wikipedia portion of the evaluation set has a substantial drop in performance with respect to target style accuracy. 
This indicates that model performance can be strongly affected by the domain of the prompt, even in setups such as ours where the prompt only contains four words. It may also indicate that the models do not learn to generalize the target style across all domains.
