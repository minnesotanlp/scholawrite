\textbf{Graph data challenges with LLMS}
LLM can not directly handle graph data, as graph data are unorganized and complex. Secondly, there is a wide range of graph-related tasks; designing a comprehensive suite of tasks and datasets for LLMs to learn on is difficult. 

\textbf{General gaps in doing reasoning tasks on graphs }
- how to convert graph to text encoding / encoding suitable for LLM
 - limited context window
 - graph structure (network complexity ;  real-world graph not considered)
 - nature of sampling method
 - no of hops considered
 - black box models 
 - choice of task - changes results
