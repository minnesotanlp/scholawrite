We use LLMs as annotators to generate labels for dataset tasks. The LLMs we chose are all open-sourced or commercially available, instruction-tuned models: Vicuna \cite{vicuna2023}, Koala \cite{koala_blogpost_2023}, Baize \cite{xu2023baize}, GPT-3.5-Turbo \cite{brown2020language}, and LLaMA 2\cite{touvron2023llama}. We create LLM task labels in two setups, \textit{intra-model} followed by \textit{inter-model}, and compare those generated labels in the inter-model setup against the original human annotations.
In the \textit{intra-model} setup, we use a combination of hyperparameters like temperature and top-p values to generate model outputs from the same model, for each model. This results in a collection of models with different response patterns, simulating how a group of humans may respond differently from each other.
