_with the \textbf{target} style, i.e. the style of the text in the paired human-machine texts. For example, if the target style is positive, the model should respond to a prompt with a positive sentiment as it did in the original human text. On the other hand, if the target style is negative, the model should respond to the prompt with a negative sentiment, although it may do so in a manner that is not identical to the original human text. We also note that when the human text is labeled with a target style, the model text is also labeled with that style. In some cases, the model may apply the target style to a text that is not similar to the human text's style. In other words, the model may apply the target style to a text that has a different formality, a different sentiment, or a different overall style. We refer to this as the ``target'' style control. We also observe that when the human text is similar to the target style, the model text is also similar to the human text's style. In this case, the model may be able to apply the target style without much control discrepancy. We refer to this as the ``target'' style control. We investigate these ideas further in Appendix\ref{sec:appendix}.
