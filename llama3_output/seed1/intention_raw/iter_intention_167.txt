\subsection{Base Models}
Our base model is LLaMA2 7B\anonymous{CITE}.

Our discriminators use the base LLaMA2 with a classification head consisting of a fully connected linear layer. We train discriminators for sentiment, formality, irony, emotion, and toxicity; for details see Table~\ref{table:discrims}\anonymous{Cite every dataset used}. We train these custom discriminators rather than using existing classifiers because classifiers with the same architecture are needed for the PPLM \anonymous{Cite..} and RL training set up. We then use these same custom discriminators in the RL fine-tuning for better comparison with PPLM.
