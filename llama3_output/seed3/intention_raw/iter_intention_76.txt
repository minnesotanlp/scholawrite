\subsection{Reward Formulations}
We explore multiple approaches to calculate a reward $R$ for a generation $x$ by combining output from the attribute discriminators $d_1, d_2,... d_n$ with target style $k_1... k_n$. Discriminators output class logits; we write the logit value for the target class as $d_i (x)_{k_i}$. We implement the following reward formulations:
\begin{itemize}
    \item \textbf{logits:} $R = \sum_{i = 0}^n \frac{1}{n} d_i(x)$ where we take the logits of the target class from the discriminator output
    \item \textbf{softmax:} $R = \sum_{i = 0}^n \frac{1}{n} \texttt{softmax}(d_i(x))_{k_i}$ where we take the softmax values of the target class from the discriminator output
    \item \textbf{binarized:} $R = \sum_{i = 0}^n \frac{1}{n} w_i$ where $w_i$ is 1 if the $\texttt{softmax}(d_i(x))_{k_i} \geq 0.5$ and $-1$ otherwise. 
    \item \textbf{calibrated:} $R = \sum_{i = 0}^n \frac{1}{n} \frac{d_i(x)}{\texttt{temperature}_{i}}$ Logit scores for are adjusted by a temperature following \citet{guo2017calibration} in order to calibrate model confidence. 
    \item \textbf{dynamic weighting:} $R = \sum_{i = 0}^n \frac{1}{n} w(1 - \texttt{softmax}(d_i(x))_{k_i})$ where $w = -\frac{\partial \texttt{log}_{d_i}}{\partial d_i(x)}$.
\end{itemize}

The motivation for including the binarized and calibrated approaches is that when discriminators are unsure or have low confidence in their prediction, the reward signal becomes noisier, potentially hampering the policy learning process. 

The motivation for including the dynamic weighting approach is that when combining rewards from multiple style discriminators, it is not always clear whether a signal from one discriminator should be prioritized over another. Our approach weighs the result from each discriminator by considering the magnitude of the discriminator's gradient with respect to the cross entropy loss for $x$ with respect to the desired style.  


\subsection{PPLM}
We extend PPLM to respond to feedback from multiple discriminators. We take a backward step in the hidden state along the gradient for all of the attribute discriminators, $d_1,..., d_n$. Specifically, the gradient is for the loss function defined by $\sum_{i = 1}^{n}d_i(x) + \lambda KL(p(x) || p(x'))$. Our modification to PPLM is outlined in Figure~\ref{fig:pplm-pipe}.
