_TestCase 2: Image Modality_
Figure \ref{fig:fig4} demonstrates our experiment using the image modality encoding with the \textsc{CORA} dataset. This encoding involves using the visual rendering of a graph to visualize nodes, edges, and other information. We assess the effectiveness of this modality by evaluating how well it facilitates node classification, which we define as the ability of the system to accurately predict node properties given node-neighbor information. Notably, this modality is amenable to future research in terms of enhancing image representation, thus enhancing the overall performance of the node classification task.
