4.1.2 \textsc{GAT}: Graph Attention Network
The GAT is a neural network whose weights are entirely dependent on the neighboring node values. It is a great tool for node classification tasks, and it has been shown to perform well on many node classification datasets \cite{velickovic2018graph}. The GAT is inspired by the traditional graph attention network (GAT) proposed in \cite{velickovic2018graph}, but we consider the adjustor weights dependent on the neighboring node values. The intuition behind the "adjx" term in the loss function is to consider the average feature of the adjacent nodes when computing the feature vector of a node. Similarly, the "adjy" term considers the average feature of the adjacent nodes when computing the feature vector of a node.

\subsubsection{Motif Modality}
The motif modality uses a motif feature vector $f$ to represent the graph. Specifically, we calculate an inner product between the motif feature vector and the graph's feature vector. This is  the motif features and the graph's features are each  and is then aggregated into a single feature vector. This process is repeated across the dataset and the results are averaged over the corresponding dataset.

\begin{equation}
    F = \frac{1}{|S|} \sum_{g \in S} \frac{f_{g}}{\sum_{g' \in S} f_{g'}}
\end{equation}

where $f_{g}$ is the motif feature vector for graph $g$ and $S$ is the collection of all graphs.
