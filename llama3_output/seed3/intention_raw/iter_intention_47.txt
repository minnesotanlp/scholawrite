\subsection{Second Order Effects}\label{sec:preference-second}

\paragraph{Hypothesis} We examine if there is a discrepancy in the distribution between human and LLM preferences augmented by training a language model classifier on each human and LLM preference, which may cause a misalignment between them. 

\paragraph{Experiment Setup} We train RoBERTa \cite{liu2019roberta} on each of the human or machine evaluations from two different preference datasets, as a reward model. We first train four different reward models from each of the above datasets and then run on machine and human preferences. For each training session, we run RoBERTa-base models for 5 epochs with AdamW optimizer of learning rate $1e-4$.
