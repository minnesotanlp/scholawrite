In this paper, we use two common methods to detect whether the texts are machine-generated. (1) We examine the structure of the documents using hierarchical discourse analysis. (2) We examine the statistical features of the documents such as formality and average shortest path length. (3) We examine the overall shape of the documents graphically.
First, we apply an existing discourse parser \citep{liu-etal-2021-dmrst} to the paired human-machine texts to perform the hierarchical discourse analysis. For a given document, the parser produces a hierarchical discourse tree that adheres to a discourse framework called the Rhetorical Structure Theory \citep{mann1987rhetorical}. We note that this hierarchical tree can be seen as a recursive hypergraph in graph theory. By converting the trees to hypergraphs, we can conduct some standard graph analyses such as computing the distribution of network motifs of varying sizes. These network motifs are small directed subgraphs with edges labeled with a discourse relation and can be considered as a building block or a ``vocabulary'' of larger graphs. We check the differences in these motif distributions and validate if they represent any meaningful patterns in discourse structures, and thus, in the ways the documents are written or generated. 

\paragraph{Formality Scorer.} We make use of a publicly available sentence-level formality scorer \citep{Babakov2023Formality} that is finetuned on a dataset for formality style transfer task, \textsc{GYAFC} \citep{rao-tetreault-2018-dear}. The model is based on a \textsc{RoBERTa-base} model \citep{liu2019roberta} and computes the score between 0 and 1 range. The formality score is computed for every text generated by humans and machines. Also, for every corresponding document graph, we count the (normalized) frequency of hyperedges for all motifs present in the graph. This is because the hyperedge relations represent higher-level abstractions of text and are thus linked to the discourse hierarchy.

\paragraph{Average Shortest Path Length.} To identify a relationship between the formality of texts and the motifs present in the graphs of  we compute the average shortest path length (ASPL) of a document graph. For linear or chain-like graphs, the ASPL tends to be relatively short, as nodes are connected in a linear fashion. In contrast, more spread-out graphs will likely have a longer ASPL due to increased distances between nodes. We then calculate the average shortest path length of a graph using a dynamic programming approach for graph extraction.

Shortest path length (SPL) is the shortest distance between any two nodes in the graph. It is also called as hop count or distance.
